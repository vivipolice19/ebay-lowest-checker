============================================
Mercari → eBay 在庫同期ツール
============================================

【概要】
Mercariの商品ページを監視し、売り切れを検出したら
eBayの在庫を自動的に0に変更するツールです。

【起動方法】
1. ZIPファイルを解凍してください
2. stock_manager.exe をダブルクリックしてください
3. GUIが起動します

【使い方】
1. 商品登録
   - Mercari URL を入力
   - eBay URL を入力
   - 仕入れ価格を入力（任意）
   - 利益率を入力（任意）
   - 「商品追加」ボタンをクリック

2. 監視開始
   - 「監視開始」ボタンをクリック
   - 30分間隔で自動チェックが始まります
   - Mercariで売り切れを検出すると、eBayの在庫を0に変更します

3. 監視停止
   - 「監視停止」ボタンをクリック

4. CSV出力
   - 「CSV出力」ボタンをクリック
   - 保存先を選択してCSVファイルを出力

【eBayログインについて】
初回のeBay在庫更新時にブラウザが開きます。
eBayにログインしてください。
ログイン情報は保存されるため、2回目以降は自動ログインされます。

【仕様】
- 最大商品数: 3000
- 監視間隔: 30分
- Mercari並列チェック数: 20
- eBay並列更新数: 5

【ログ】
logs/app.log にログが記録されます。
問題が発生した場合はこのファイルを確認してください。

【注意事項】
- インターネット接続が必要です
- eBayアカウントが必要です
- 初回起動時にブラウザが自動ダウンロードされる場合があります

【フォルダ構成】
stock_manager.exe  - メインプログラム
database/          - データベースフォルダ
logs/              - ログフォルダ
README.txt         - このファイル
